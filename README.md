# Autobacs PR Training Pre-Post Survey App

## Run Local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
1. Login to Vercel
2. Import GitHub repository `tanikornc32-ui/autobacs-training`
3. Framework preset: Vite
4. Deploy

## Features
- Pre/Post survey
- 10-question scoring
- Dashboard charts
- Export XLSX
- Export PPTX
- Autobacs orange theme
